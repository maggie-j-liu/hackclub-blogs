## Development

First, fill out the env file:

```bash
cp .env .env.local
# and add the necessary credentials
```

Install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
